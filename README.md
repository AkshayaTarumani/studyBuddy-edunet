AI-Powered Study Buddy



AI-Powered Study Buddy is an intelligent learning assistant designed to help students understand complex topics more easily. The application allows users to upload PDF study materials and ask questions related to the content. Using Google's Gemini AI model, the system provides simplified explanations, concise summaries, quiz questions, and flashcards to enhance learning and revision.

Features

- Upload PDF study materials
- Extract and analyze text from PDF documents
- Generate simple explanations for complex concepts
- Create concise summaries for quick revision
- Generate quizzes for self-assessment
- Create flashcards for effective learning
- Interactive and user-friendly interface using Streamlit

Technologies Used

- Python
- Streamlit
- Google Gemini API
- PyPDF2
- GitHub

-------How It Works ?-------------------------

1. Upload a PDF document.
2. The system extracts text from the uploaded file.
3. Enter a question or select a learning task.
4. Gemini AI processes the content.
5. The application generates explanations, summaries, quizzes, or flashcards.
6. Results are displayed instantly to the user.

#####Installation

Clone the Repository

git clone <repository-url>
cd ai-powered-study-buddy

Install Dependencies

pip install -r requirements.txt

Configure Gemini API Key

Create a ".env" file and add your API key:

GEMINI_API_KEY=your_api_key_here

Run the Application

streamlit run app.py
